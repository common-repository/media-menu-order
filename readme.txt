Add a Menu Order field to Media Library.

== Description ==

A very simple plugin that adds a Menu Order field to Media Library editing interfaces, allowing you to assign menu order to images and attachments.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. There's no step 3!
